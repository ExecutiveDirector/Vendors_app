# Vendor App (Flutter)

A vendor/operator mobile app for managing orders, outlets, inventory, riders, promotions, analytics, notifications, and support.
Maps to your Sequelize entities: vendors, vendor_outlets, vendor_inventory, vendor_analytics, products, product_categories, orders, order_items, riders, delivery_assignments, promotions, reviews, notifications, support_tickets, transactions, and more.

## Quick Start
```bash
flutter pub get
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:3000
```
